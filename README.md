# 🐍 Snake Game (C / SDL2)

A classic **Snake game** developed in **C** using the **SDL2 library** and SDL_ttf.

The player controls a snake, eats apples to grow, avoids rocks, and can earn bonus points with a rabbit that appears during gameplay.

---

## 🎮 Features

- Smooth snake movement
- Snake growth system
- Score system
- Obstacles (rocks)
- Bonus rabbit spawning
- Main menu (Play / Quit)
- Game Over screen
- Collision detection (walls, body, rocks)

---

## 🕹️ Controls

- ⬆️ Up Arrow: move up  
- ⬇️ Down Arrow: move down  
- ⬅️ Left Arrow: move left  
- ➡️ Right Arrow: move right
